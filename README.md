# 📊 Análise de E-commerce - Olist

## 🎯 Objetivo
Analisar fatores que impactam a satisfação do cliente no e-commerce, com foco em tempo de entrega e experiência logística.

## 📁 Dataset
Base pública Olist (Kaggle) com 99 mil pedidos entre 2016 e 2018.

## 🔍 Análises realizadas
- Limpeza e tratamento de dados
- Análise de outliers
- KPIs de negócio
- Relação entre tempo de entrega e avaliação
- Análise geográfica

## 📊 Principais insights
- Crescimento do volume de pedidos e receita
- Redução do tempo de entrega ao longo do tempo
- Relação direta entre atraso e insatisfação

## 🚀 Conclusão
A eficiência logística é o principal fator de impacto na satisfação do cliente, sendo essencial para sustentar o crescimento do negócio.
